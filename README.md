# Despite The Miles

I wasn't able to be with my family on christmas physically this year, so I decided to do what I do best, build something they can always look back on.

Love ya'll <3

This is an interactive 3D holiday scrapbook with a page-flipping book, ambient audio, and a gentle snowfall backdrop.

## Features
- 3D animated book with page-flip interactions
- Clickable UI for navigating pages
- Ambient background music with page-flip SFX
- Snow particle system in the scene background

## Tech Stack
- React + Vite
- @react-three/fiber + @react-three/drei (Three.js)
- Tailwind CSS
- Jotai

## Getting Started
1) Install dependencies: `npm install` (or `yarn`)
2) Run dev server: `npm run dev` (or `yarn dev`)

## Project Notes
- Photo textures live in `public/textures`.
- Audio assets live in `public/audios`.
- Scene and effects are in `src/components/Experience.jsx`.
